# Configuration reference

This page is a repository-level entry point for configuration documentation. Package-specific configuration remains in package, runtime, registry, CLI, and operations docs.

## Configuration categories

- Runtime/server options.
- Registry options.
- CLI flags.
- MCP bridge settings.
- Environment variables.
- Release and CI configuration.

## Stability policy

Configuration keys used by external users should be documented, tested where practical, and handled under the deprecation policy when changed.

## Follow-up

As package APIs stabilize, this page should link to generated or package-specific configuration reference pages.
