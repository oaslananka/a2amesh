# Compatibility reference

This page summarizes compatibility expectations and points to deeper protocol/package docs.

## Compatibility dimensions

- Node.js and pnpm versions from `package.json` engines.
- A2A protocol versions and compatibility matrices.
- CLI command behavior.
- Published package exports.
- Registry API behavior.
- MCP tool schema behavior.

## Change policy

Compatibility-sensitive changes should include tests, release notes, and migration guidance when required.
