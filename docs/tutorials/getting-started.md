# Getting started tutorial

This tutorial points new users to the fastest safe path through the repository.

## Prerequisites

- Node.js matching the repository `engines` range.
- Corepack enabled.
- pnpm matching `packageManager` in `package.json`.

## Steps

1. Clone the repository.
2. Run `pnpm run setup`.
3. Run `pnpm run build`.
4. Run `pnpm run test:unit`.
5. Read `docs/quickstart.md` for runtime usage.
6. Read `docs/examples.md` for example scenarios.
