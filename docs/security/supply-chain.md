# Supply Chain

Supply-chain checks include lockfile verification, generated-artifact scans, secret scans, audit checks, CodeQL, Dependency Review, OpenSSF Scorecard, actionlint, zizmor, package dry-runs, and public-surface inventories.

## Release package evidence

Release package evidence is documented in [Package Verification](../release/package-verification.md). Keep SBOM, checksum, provenance, npm dist-tag, and registry parity evidence with each GitHub Release.
